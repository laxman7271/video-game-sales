# 🎮 Video Game Sales Prediction using Linear Regression

## 📌 Project Overview

This project predicts **global video game sales** using a **Linear Regression Machine Learning model**.
The dataset contains information about video games such as platform, genre, publisher, critic score, and user score.

The goal of this project is to analyze video game sales data and build a model that can **predict the global sales of a game** based on its features.

---

## 📂 Dataset

Dataset used: **Video Game Sales Dataset**

Source: Kaggle
https://www.kaggle.com/datasets/rush4ratio/video-game-sales-with-ratings

File used:

```
Video_Games_Sales_as_at_22_Dec_2016.csv
```

Important Columns:

* Name
* Platform
* Year_of_Release
* Genre
* Publisher
* Critic_Score
* User_Score
* Global_Sales

Target Variable:

```
Global_Sales
```
